== Description ==

Set country and IP to block your website. You can also set IP address to unblock for any special IP Address.


== Installation ==

1. Upload 'block-country' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the Block Country 
4. You can choose your country, IP address to block and enter your message. You can aslo set IP address to unblock for blocked country.
5. Click on "Submit" button.
